# How to markdown

Basic Syntax:

## Headings:
'# Heading 1'
'## Heading 2'
'### Heading 3'
And so on...

## Bold:
'**bold text**'
Italics:
'*italic text*'

## Bold and Italics:
'***bold and italic text***'

## Unordered lists:
'- Item 1'
'- Item 2'

## Ordered lists:
'1. Item 1'
'2. Item 2'

## Links:
'[Link text](URL)'

## Images:
'![Alt text](image URL)'

## Blockquotes:
'> Blockquote'
